Setup

This section specifies the Python version and the required packages.

Python Installation

Please download the Python installer from here. We are using the 3.9.2 version.

Requirements

The followings are the required Python packages for the Feedback Forum.

  • Flask==1.1.2
  • Flask-Bootstrap==3.3.7.1
  • Flask-Login==0.5.0
  • Flask-Mail==0.9.1
  • Flask-Migrate==2.7.0
  • Flask-SQLAlchemy==2.4.4
  • Flask-WTF==0.14.3

After Python installation, install all the packages through the terminal with the instruction.

pip install <package name>

Plug in each package name listed above.
