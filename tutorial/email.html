Email Sending

Email sending is a auxiliary function utilized to send the token to the client.

We create a file email.py under the folder app, and add content as this code. This piece of code is referred to the Chapter 6 of “Flask Web Development, 2nd Edition”[1]. It provides a simple email function but suitable for our need.

send_async_email

This function triggers the email sending under the context of the app. It would be executed in a thread.

send_email

This function composes the email content and create a thread to send the email to the client.

Gmail Account Settings

This tutorial is using Google Mail as an example in the developing environment. The configurations related to the email sending functionality is set for Google mail account. Further more, you need to lower the security level of your Google account.

Please remember to reset the security level after completing the tutorial.
